# 第十章：生产部署与交付

本章将讲解如何把一个“能跑起来的项目”交付成“可以上线、可以维护、出了问题也能排查”的生产系统。

::: info 🧭 章节衔接
走到这里，前后端功能本身已经基本闭环；这一章处理的是最后也是最现实的问题：如何把它稳定地带到生产环境。  
Docker、Nginx、性能优化和故障排查，会把整套教程从“开发完成”推进到“真正可交付”。
:::
