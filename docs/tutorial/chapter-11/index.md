# 第十一章：Vue 3 管理后台

本章将使用 Vue 3 + TypeScript 开发管理后台前端，把前面已经完成的认证、权限、内容管理和统计能力真正呈现在用户面前。

::: info 🧭 章节衔接
前面 1-10 章已经把后端接口、认证权限、业务能力和系统能力逐步补齐；这一章开始把这些能力真正呈现在用户面前。  
你会看到登录、菜单、页面布局和接口封装，如何把前面的后端设计完整串成一个可操作的管理后台。
:::

## 本章导读

这一章不只是“把接口接上去”，而是把前面 1-10 章完成的后端能力真正变成可操作、可浏览、可管理的后台界面：
- Vue 3 项目初始化
- API 请求封装
- 登录认证与路由控制
- 布局、菜单与后台首页

## 学习目标

完成本章后，你将：
- ✅ 搭建 Vue 3 管理后台项目
- ✅ 封装 API 请求与统一异常处理
- ✅ 实现登录认证和路由守卫
- ✅ 渲染菜单与基础页面
- ✅ 理解前后端权限与菜单的对接方式
