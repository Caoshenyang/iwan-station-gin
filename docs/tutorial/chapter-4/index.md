# 第四章：用户认证系统

用户认证是后台管理系统的基础功能。本章我们将实现基于 JWT 的用户认证系统。

::: info 🧭 章节衔接
前面三章解决了“项目怎么组织、服务怎么跑起来”的问题；从这一章开始，我们正式进入后台系统最核心的安全主线。  
这一章先完成“你是谁”的认证问题，下一章再继续解决“你能做什么”的授权问题。
:::

## 本章导读

认证系统是整个系统的安全基础。我们将实现：
- JWT Token 生成与验证
- 用户模型设计
- 登录/注册接口
- 认证中间件

## 学习目标

完成本章后，你将：
- ✅ 理解 JWT 认证原理
- ✅ 设计用户数据模型
- ✅ 实现登录/注册功能
- ✅ 编写认证中间件
- ✅ 掌握密码加密技术
