# 第六章：RBAC 权限与菜单模型

这一章解决的是“你能做什么”。只有把角色、权限和菜单三者的关系设计清楚，后台系统才算真正具备安全边界。

::: info 🧭 章节衔接
上一章我们完成了用户身份认证；这一章继续补上授权体系。  
学完这一章后，系统就不只是“能登录”，而是能够根据角色和权限控制菜单、接口和操作范围。
:::
