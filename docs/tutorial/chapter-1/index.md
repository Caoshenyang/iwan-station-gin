# 第一章：课程介绍与准备

欢迎来到《Iwan Station Gin - 从零构建生产级全栈项目》！本教程将带你完整开发一个包含 Go + Gin 后端和 Vue 3 前端的生产级管理系统。

## 本章导读

本章将帮助你做好学习前的准备工作，包括：
- 了解课程目标和项目背景
- 搭建完整的开发环境
- 初始化项目结构
- 快速掌握 Go 基础（针对 Java 开发者）

## 学习目标

完成本章后，你将：
- ✅ 了解项目的技术栈和功能模块
- ✅ 搭建好 Go、PostgreSQL、Redis 开发环境
- ✅ 创建项目基础目录结构
- ✅ 理解 Go 与 Java 的核心差异
