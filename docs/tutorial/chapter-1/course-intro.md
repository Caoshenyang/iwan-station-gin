# 📚 1.1 课程概述

::: tip 💡 阅读建议
这一页适合先建立全局地图：你会学到什么、项目长什么样、整条路线会走到哪里。第一次读不需要记住所有细节，先抓住课程结构即可。

## 🎯 学习目标

完成本课程后，你将能够：

- ✅ 从零开始构建生产级 Go 后台 API
- ✅ 实现 JWT 用户认证系统
- ✅ 设计并实现 RBAC 权限管理系统
- ✅ 使用 GORM 进行数据库操作
- ✅ 集成 Redis 进行缓存优化
- ✅ 开发 Vue 3 + TypeScript 管理后台
- ✅ 使用 Docker 部署完整应用

## 👥 适合人群

本课程适合：

- 后端开发者学习 Go
- Java 开发者转型 Go
- 全栈开发者学习完整项目
- 希望看到生产级代码的开发者

## 🛠️ 前置知识

### 必备知识
- HTTP/REST API 基础
- SQL 数据库基础
- JavaScript/TypeScript 基础

### 工具要求
- Go 1.21+
- PostgreSQL 15+
- Redis 7.0+
- Node.js 20+（前端开发）
- Docker（可选）

## 🧩 项目介绍

### ✨ Iwan Station

一个功能完整的后台管理系统：

**后端功能：**
- 🔐 JWT 用户认证
- 👥 用户管理
- 🎭 角色权限（RBAC）
- 📁 分类管理
- 🏷️ 标签管理
- 📝 文章管理
- 📊 数据统计
- ⚙️ 系统配置

**前端功能：**
- 🎨 Vue 3 + Naive UI
- 📱 响应式设计
- 🔍 搜索过滤
- 📈 数据可视化
- 🎯 动态路由菜单

## 📌 学习方式

本课程采用**实践优先**的方式：

1. **概念讲解** - 理解"为什么"
2. **代码实现** - 掌握"怎么做"
3. **设计决策** - 学习权衡取舍
4. **Go vs Java** - 关联已有知识
5. **动手实践** - 构建真实功能

## 🏗️ 项目结构

```
iwan-station-gin/
├── server/               # Go 后端
│   ├── cmd/server/       # 入口文件
│   ├── internal/         # 私有代码
│   │   ├── api/          # HTTP 处理器
│   │   ├── service/      # 业务逻辑
│   │   ├── repository/   # 数据访问
│   │   ├── model/        # 数据模型
│   │   ├── middleware/   # 中间件
│   │   └── pkg/          # 工具函数
│   └── config/           # 配置文件
├── admin/                # 管理后台前端（教程主线）
├── docs/                 # VitePress 文档
└── web/                  # 门户前端（可选扩展）
```
