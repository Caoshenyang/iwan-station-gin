# 第五章：用户体系与 JWT 认证

认证是后台系统的第一条安全主线。这一章先解决“你是谁”，把用户身份、密码安全、Token 生命周期和认证中间件一步步搭完整。

::: info 🧭 章节衔接
上一章我们已经把 Web 层的公共约定定清楚了；这一章开始进入真正的安全主线。  
学完这一章后，系统会先具备稳定的登录认证能力，下一章再继续补上授权与菜单控制。
:::

## 本章导读

这一章会完整建立认证链路：
- 用户模型设计
- 密码加密与校验
- JWT 生成、解析与续期
- 登录接口与认证中间件

## 学习目标

完成本章后，你将：
- ✅ 设计用户模型与认证相关字段
- ✅ 掌握密码安全存储方案
- ✅ 理解 JWT 的签发与校验流程
- ✅ 实现登录接口与认证中间件
- ✅ 为下一章授权体系打好身份基础
